Sistem Bileşenleri
📡
YouTube Platform
Veri kaynağı ve dağıtım kanalı. Comments API ile yorumlar çekilir, Members API ile üyeler kontrol edilir, Comments.insert ile video yanıtlar gönderilir. Günlük 10.000 ünite kota.
⚡
n8n Orchestrator
Tüm otomasyonu yöneten merkez beyin. 4 ana workflow çalıştırır: Üye algılama (15dk), Yorum tarama (10dk), Onay gönderim (realtime), Analitik toplama (günlük). Self-hosted Docker.
🧠
AI Services
GPT-4o-mini senaryo üretir (~$0.90/ay). ElevenLabs ses klonlar ($22/ay). HeyGen video üretir ($89/ay). OpenAI Moderation güvenlik kontrol eder (ücretsiz).
🗄️
Supabase
PostgreSQL veritabanı (6 tablo), Auth (Google OAuth + YouTube OAuth), Storage (video dosyaları), Realtime (canlı dashboard güncellemeleri), Edge Functions. RLS ile veri izolasyonu.
🖥️
Creator Dashboard
Next.js 15 + shadcn/ui. Tinder-style swipe onay kuyruğu, üye profilleri, analitik dashboard, ayarlar. Vercel'de deploy. Cursor IDE ile geliştirme.
💰
Maliyet Özeti
Toplam MVP: $113-158/ay. HeyGen $89 + ElevenLabs $22 + OpenAI $1 + Hosting $0-25 + Supabase $0-25. Breakeven: 3 ödeyen creator.

Adım Detayları — Workflow 1
⏰
Tetikleyici
n8n Cron node, her 15 dakikada bir çalışır. Gece saatlerinde (00:00-06:00) 30 dakikaya düşürülür (kota tasarrufu).
📡
YouTube Members API
members.list ile tüm aktif üyeleri çeker. Supabase'deki mevcut üyelerle karşılaştırılarak yeni üyeler tespit edilir. Kota: 1 ünite/istek.
🤖
Senaryo Üretimi
GPT-4o-mini, creator'ın persona prompt'u ve knowledge base'i ile kişiselleştirilmiş 40-80 kelimelik hoş geldin senaryosu üretir. temperature: 0.8 ile doğal çeşitlilik sağlanır.
🎙️
Ses Üretimi
ElevenLabs eleven_multilingual_v2 modeli ile creator'ın klonlanmış sesi kullanılarak MP3 üretilir. Ortalama süre: 2-5 saniye işlem.
🎭
Video Üretimi
HeyGen API avatar + ses dosyasını birleştirerek 720x1280 (9:16) video üretir. Üretim süresi: 60-120 saniye. 10sn aralıkla polling ile tamamlanma kontrol edilir.
💾
Kayıt & Bildirim
Video video_responses tablosuna pending_approval statüsüyle kaydedilir. Creator'a push bildirim + dashboard badge güncellenir.
WF1 Çalışma Süresi
~3 dk / üye
Senaryo: 2sn + TTS: 3sn + Video: 90sn + Polling: 30sn

Adım Detayları — Workflow 2
💬
Yorum Çekme
commentThreads.list ile son yorumlar çekilir. maxResults: 100 ile her sorguda 100 yorum. Üye rozeti kontrolü için authorChannelId ile members tablosu eşleştirilir.
🧠
Niyet Sınıflandırma
GPT-4o-mini JSON formatında sınıflandırma yapar: intent (soru/övgü/şikayet), priority (1-10), sentiment (-1 to 1), video_worthy (true/false). temperature: 0.1 ile tutarlılık.
📊
Öncelik Puanlama
n8n Code node ile hesaplama: Üyelik süresi (+2), soru içeriyor (+3), negatif duygu (+2), uzun yorum (+1), ilk yorum (+2). Toplam ≥7 ise video yanıt tetiklenir.
⚖️
Karar Noktası
priority ≥ 7 AND video_worthy = true → Video yanıt pipeline'ı başlatılır. Aksi halde metin taslak olarak kaydedilir (gelecek fazda metin yanıt özelliği).
WF2 Günlük Kota
~96 ünite
144 polling (10dk) × 1 ünite = 144, üye filtresi sonrası ~96 etkin sorgu

Adım Detayları — Workflow 3
⚡
Realtime Trigger
Supabase Realtime subscription ile video_responses.status değişimini dinler. Creator dashboard'dan swipe ile onay geldiğinde anında tetiklenir.
⏳
Jitter Delay
YouTube'un bot algılamasını engellemek için 5-30 dakika rastgele bekleme. Formül: Math.random() × 25 + 5. Kota %80'i aşarsa 30-60dk'ya çıkarılır.
📤
YouTube Gönderim
comments.insert ile yorum yanıtı gönderilir. Welcome videoları için topluluk postu veya yorum, Reply videoları için orijinal yoruma yanıt. 50 ünite/istek kota maliyeti.
📊
Durum Güncelleme
video_responses → sent, members → welcome_video_sent + total_videos_received güncellenir. Analytics log'a tüm metrikler kaydedilir.
WF3 Kota Maliyeti
50 ünite / gönderim
Günde 50 video gönderim = 2.500 ünite (toplam kotanın %25'i)

YouTube Bağla (2dk)

Ses Klonla (3dk)

Hata Yönetim Stratejisi
📡
YouTube Hataları
401: Token refresh akışı başlatılır, başarısızsa creator'a yeniden bağlanma bildirimi. 403: Kota aşımı, sonraki cron'a ertelenir. 5xx: Exponential backoff (5-15-45sn).
🎭
HeyGen Hataları
Video üretim hatası: 3 retry (farklı arka plan). Timeout (>5dk): İptal + yeniden kuyruğa al. Rate limit: 60sn bekleme. En kritik hata noktası — video üretilemezse metin fallback devreye girer.
🎙️
ElevenLabs Hataları
Karakter limiti: Plan upgrade uyarısı. Voice bulunamadı: Yeniden klonlama. 5xx hatası: Edge TTS'e fallback (ücretsiz ama kalite düşük, geçici çözüm).
🤖
OpenAI Hataları
Rate limit: 10sn bekleme. Content filter: Farklı prompt ile tekrar. 5xx: Claude Haiku'ya fallback. En düşük risk — OpenAI hataları nadiren pipeline'ı durdurur.
🛡️
Ultimate Fallback
Tüm retry'lar başarısızsa → Metin yanıt gönder. Video yerine kişiselleştirilmiş metin mesaj. Creator'a "video üretilemedi" bildirimi. Hiçbir üye yanıtsız kalmaz.
